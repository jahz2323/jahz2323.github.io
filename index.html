Master's Project: Evaluation of Fourier Convolutions

Final Year Project targeted to speedup expensive Convolution utilising the Fourier Domain

Investigated and Profilled several methods of Convolution under CPU and GPU backends and Parallel frameworks @CUDA and WEBGPU.

Developed three customised CUDA kernels - Direct Convolution, Fast-Fourier Convolution and FFT-Overlap and Add

Created customised test-suites of stochastic and deterministic inputs and verified the results measuring MSE, PSNR and SSIM.

40% Speedup over Direct Convolution when using Fourier-based Convolution when using large kernel and inputs

Fourier-based Convolution increased memory overhead by +95% but alternative FFT method via FFT-Overlap and Add reduced consumption to only 20%

MSE for both spectral methods maintained less than 10pow(-18) error with respect to direct convolution

Stack used:

  • CUDA
  • WEBPGU
  • DOCKER
  • OpenCV
  • C++ and Python
